# cases/

Demo analysis scripts organized by dataset, used as reading material for an intro data science class.

| Entry | Purpose |
|-------|---------|
| `seaborn_diamonds/` | EDA, regression, and dimensionality reduction on diamond price data |
| `seaborn_healthexp/` | EDA and linear regression on health expenditure vs. life expectancy |
| `seaborn_mpg/` | EDA plus Ridge/Lasso regularization and bias-variance tradeoff demos |
| `seaborn_penguins/` | EDA and classification pipeline comparison (baseline → trees → model zoo) |
| `seaborn_planets/` | EDA and anomaly detection with Isolation Forest on exoplanet data |
| `seaborn_tips/` | EDA and single- vs. multi-feature regression on restaurant tips data |
| `sklearn_diabetes/` | EDA and polynomial regression on the sklearn diabetes dataset |
| `sklearn_kddcup99/` | EDA, anomaly detection, and binary/multi-class classification on network intrusion data |
| `synthetic_data/` | Isolation Forest demo on synthetically generated data |
| `utils/` | Shared helpers: resampling, model registry, regression utilities |
| `warmup.py` | Short warm-up script introducing NumPy basics |
