# Data Science and Machine Learning: Code Reading Exercises

Data science, like programming, can increasingly be streamlined using strong AI coding support.
As a consequence, memorizing exact syntax becomes less important than understanding it.
What's more important is **understanding concepts** and the general **data science process**, as well as recognizing all of it in code.

This repo is a resource with various ready-made analyses for seaborn datasets (they'll be added gradually).
